Integration of GitHub and Argo Events

This application demonstrates a comprehensive integration setup:

  • GitHub Repository: Source code management and version control.
  • Node.js Application: Simple HTML page.
  • Docker: Containerizing the application.
  • Argo Events: Event-driven workflows in Kubernetes.
  • Argo Workflows: Includes steps to clone the repository, build the Docker image, push it to DockerHub, and deploy the application using Kubernetes.
  • Sealed Secrets: Used to store private keys for the GitHub repository.

This setup showcases how various components work together to create a seamless GitOps deployment pipeline.
